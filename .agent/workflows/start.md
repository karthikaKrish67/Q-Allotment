# Starting the SmartQ Allotment System

Follow these steps to start the application:

// turbo-all

1. Start the backend server:
   ```
   cd c:\Users\lenovo\Desktop\Allotment\backend
   npm run dev
   ```

2. Start the frontend server (in a new terminal):
   ```
   cd c:\Users\lenovo\Desktop\Allotment\frontend
   npm run dev
   ```

3. Open the application in browser: http://localhost:5173

4. Login credentials:
   - Username: admin
   - Password: admin123

## Quick Start (Alternative)
You can also double-click the `START_SERVERS.bat` file in the Allotment folder to start everything automatically.
